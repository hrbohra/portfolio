<Plain>
Three complete applications, built independently in a two-week burst. A book chatbot you can
hand any novel to, which answers questions without spoiling chapters you have not reached. An
app that repaints your photos in the style of famous artworks. And an analysis tool that
reads thousands of text records, works out the mood of each, and writes a plain-English
report with charts.
</Plain>
<Eng>
One: a spoiler-aware book Q&A chatbot. Streamlit, PDF and DOCX ingestion, spaCy entity
extraction, sentence-transformers retrieval scoped by reading position, SQLite conversation
memory. Two: a neural style-transfer web app, React frontend with a Python Flask inference
backend. Three: a GPT-Neo 1.3B analysis tool running sentiment and time-series analysis over
a real dataset, generating Matplotlib visualisations and an automated business-readable
report.
</Eng>
